* How to use

Add =nixmacs= to your =flake.nix= inputs:

#+begin_src nix
inputs = {
  nixpkgs.url = "github:NixOS/nixpkgs";

  nixmacs = {
    url = "github:yvnth/nixmacs";
    inputs.nixpkgs.follows = "nixpkgs";
  };
};
#+end_src
