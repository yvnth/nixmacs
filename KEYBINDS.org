#+TITLE: Emacs Keybindings
#+AUTHOR: yvnth
#+options: toc:nil

** Vanilla Emacs - Quitting
| Key     | Description                          |
|---------+--------------------------------------|
| C-x C-c | Save buffers (asks) and quit Emacs   |
| C-g     | Cancel current command or prompt     |
| C-x k   | Close current buffer                 |
| C-z     | Suspend/iconify Emacs (terminal/GUI) |

** Vanilla Emacs - Files
| Key       | Description                               |
|-----------+-------------------------------------------|
| C-x C-f   | Open a file                               |
| C-x C-v   | Open another file, killing current buffer |
| C-x C-s   | Save current buffer                       |
| C-x C-w   | Save buffer as a different file           |
| C-x s     | Save all buffers, asking for each         |
| C-u C-x s | Save all buffers, no asking               |
| C-x C-q   | Toggle read-only mode                     |
| C-x i     | Insert contents of a file at point        |

** Vanilla Emacs - Buffers
| Key         | Description              |
|-------------+--------------------------|
| C-x b       | Switch to another buffer |
| C-x C-b     | List all buffers         |
| C-x k       | Kill a buffer            |
| C-x <right> | Move to next buffer      |
| C-x <left>  | Move to previous buffer  |

** Vanilla Emacs - Navigation
| Key       | Description                     |
|-----------+---------------------------------|
| C-a       | Go to start of line             |
| C-e       | Go to end of line               |
| C-k       | Kill from cursor to end of line |
| M-<       | Move to top of buffer           |
| M->       | Move to bottom of buffer        |
| M-f       | Move forward one word           |
| M-b       | Move backward one word          |
| M-g g <n> | Go to line number n             |
| M-g c <n> | Go to character position n      |

** Vanilla Emacs - Bookmarks
| Key     | Description                     |
|---------+---------------------------------|
| C-x r m | Add current buffer to bookmarks |
| C-x r b | Open a buffer from bookmarks    |
| C-x r l | List bookmarks                  |

** Vanilla Emacs - Search
| Key   | Description                                       |
|-------+---------------------------------------------------|
| C-s   | Forward search / jump to next match               |
| C-r   | Backward search / jump to previous match          |
| M-%   | Query-replace                                     |
| C-g   | Exit search                                       |
| M-s . | Find symbol under cursor                          |
| M-s w | Find symbol under cursor, matching separators too |
| M-s o | List all matching lines (occur)                   |

** Vanilla Emacs - Select, Copy, Cut, Paste
| Key   | Description                |
|-------+----------------------------|
| C-_   | Undo                       |
| C-x u | Redo                       |
| C-SPC | Begin selection (set mark) |
| C-g   | Cancel selection           |
| C-x h | Select the whole buffer    |
| M-w   | Copy                       |
| C-y   | Paste (yank)               |
| C-w   | Cut (kill region)          |

| M-l | Make word lowercase                 |

** Vanilla Emacs - Dired (file manager)
| Key     | Description                                       |
|---------+---------------------------------------------------|
| C-x d   | Open Dired on a directory                         |
| C-x 4 d | Open Dired in another window                      |
| C-x 5 d | Open Dired in another frame                       |
| f       | Open file/dir at point, in current window         |
| o       | Open file/dir at point, in another window         |
| +       | Create a new directory                            |
| (       | Toggle showing file details                       |
| C-x C-q | Make listing editable (C-c C-c to commit renames) |
| w       | Copy filename (relative) to kill ring             |
| C-0 w   | Copy full filename path to kill ring              |
| m       | Mark file/directory                               |
| u       | Unmark file                                       |
| t       | Toggle marks on all files                         |
| X or !  | Run shell command on file at point                |
| M-!     | Run shell command on marked files                 |
| &       | Run async shell command on file(s)                |
| s       | Change sort order                                 |
| D       | Delete file                                       |
| g       | Refresh/revert Dired buffer                       |
| R       | Rename/move file                                  |
| y       | Show file type                                    |
| q       | Quit Dired buffer                                 |

** Vanilla Emacs - Misc Useful Commands (M-x)
| Command                    | Description                                                 |
|----------------------------+-------------------------------------------------------------|
| delete-trailing-whitespace | Strip trailing whitespace/^M from buffer                    |
| tabify / untabify          | Convert spaces to tabs / tabs to spaces                     |
| whitespace-mode            | Toggle visualization of whitespace characters               |
| linum-mode                 | Toggle line numbers (legacy; display-line-numbers-mode now) |
| revert-buffer              | Reload buffer from disk                                     |
| auto-revert-mode           | Auto-reload buffer when file changes on disk                |
| auto-revert-tail-mode      | Follow a growing file, like tail -f                         |
| rename-buffer              | Rename the current buffer                                   |
| auto-fill-mode             | Auto-wrap paragraphs as you type                            |
| shell-command (M-!)        | Run a shell command, show output in a buffer                |
| async-shell-command (M-&)  | Run a long-running/async shell command                      |
| compile                    | Run a compile command (press g to re-run)                   |
| find-name-dired            | Find files by name pattern, show results in Dired           |
| find-grep-dired            | Find files whose contents match a grep pattern              |
| rgrep                      | Search file contents recursively with grep                  |
| locate                     | Search files via the locate database                        |
